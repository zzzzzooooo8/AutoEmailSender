# v2.1.6

## 更新内容

### 新增功能

- 新增桌面端更新公告弹窗，发现新版本时可以直接查看完整更新内容。
- 更新公告支持 Markdown 渲染，正文区域固定高度并可内部滚动，内容较长时不会撑高弹窗。
- 发布流程新增公告草稿生成和正式发布前校验，便于先润色更新说明，再发布新版本。

### 体验优化

- 优化更新通知去重逻辑，减少重复提醒。
- 优化首页加载态与学校 / 学院筛选的联动体验。

### 问题修复

- 修复页面快照缓存和候选补全 worker 的稳定性问题。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.1.6.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
