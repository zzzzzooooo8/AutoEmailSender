# v2.1.7

## 更新内容

### 新增功能

- 批量草稿支持后台生成，任务开始后可以继续使用其他页面，并在任务中心查看生成进度、运行设置和当前状态。
- 批量草稿任务支持取消与恢复，适合处理耗时较长或需要中途调整的生成流程。
- 任务中心新增“当前任务 / 回收站”切换，批量邮件、教师抓取和匹配分析任务都可以进入回收站统一查看。
- 设置页新增批量草稿生成运行时字段，为并发生成、状态展示和任务恢复提供配置基础。

### 体验优化

- 批量任务现在会区分“正在生成草稿”和“模板直通”状态，任务列表里的进度含义更清晰。
- 手动中止批量邮件任务后，详情页会继续在“还未发送给”里展示未发送导师，并标记为“已取消”，方便后续继续联系。
- 批量草稿生成过程会记录 token 使用情况，便于后续排查生成成本和任务执行结果。
- 首页高级筛选选项增加间距，筛选区域在内容较多时更容易阅读和操作。
- 任务中心回收站切换按钮调整布局和配色，当前视图状态更容易识别。

### 问题修复

- 修复回收站任务仍可能被调度器继续处理的问题，已移入回收站的任务不会再参与后台调度。
- 修复其他设置里“批量邮件 LLM 草稿并发数”在默认状态下可能显示 `undefined` 的问题。
- 修复模板占位符编辑时光标跳转的问题，编辑较长模板时输入位置更稳定。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.1.7.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
