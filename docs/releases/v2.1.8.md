# v2.1.8

## 更新内容

### 新增功能

- 批量草稿生成支持按模板运行重写，能够把模板中的片段替换成更贴近目标场景的内容。
- 运行重写现在支持文本模板，并会预估草稿 token 用量，方便提前判断任务规模。

### 体验优化

- 创建任务页接入了 HTML 编辑器，并对目标导师列表做了分页展示，长列表浏览和编辑更顺手。
- 诊断日志导出分组进行了优化，排查问题时更容易定位。
- 发送设置里遗留的 dry-run 相关内容已清理，界面信息更简洁。
- 更新公告弹窗的居中和 Markdown 渲染已修复，公告内容展示更稳定。

### 问题修复

- 修复模板占位符重复时仍可能写入的问题，避免生成结果里出现无效 token。
- 修复 responses payload 与空列表富文本的兼容问题，降低后端解析异常。
- 修复批量邮件模板立即发送未自动发送的问题。
- 修复 LLM 改写模板时富文本格式可能被破坏的问题。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.1.8.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
