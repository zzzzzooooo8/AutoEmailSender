# v2.2.0

## 更新内容

### 新增功能

- 草稿生成启用锚点化模板改写，系统会按模板片段定位并写回改写结果，减少整段替换带来的格式偏移。
- 教师抓取任务支持“部分已导入”状态，部分导入后仍可继续审核和处理剩余候选。
- 桌面端新增后端启动阶段和就绪状态反馈，启动过程中的状态变化更清楚。
- 前端支持真实节假日日期标识，相关日期展示更贴近实际日历。
- 工作区会提示草稿生成消耗，便于在运行前了解任务成本。

### 体验优化

- 草稿模板改写补充段落连续上下文，提升改写结果与原模板前后内容的衔接度。
- 抓取任务候选审核选择规则更严格，减少误选无关候选的情况。
- 批量邮件会更明确地区分定时模板的自动发送状态。
- 桌面端启动提示改为更友好的用户文案，身份信息保存会等后端就绪后再开放。
- 前后端测试运行速度已优化，日常验证和发布检查更快。

### 问题修复

- 修复桌面端后端进程残留、重复启动以及关闭开发环境时后端未清理的问题。
- 修复桌面端启动状态短暂异常时可能导致前端误判的问题。
- 修复抓取日志和草稿审核状态显示不一致的问题。
- 修复历史“部分导入”抓取任务状态不准确的问题。
- 修复任务中心仍显示待生成草稿处理入口的问题。
- 修复首页导师看板筛选条件在返回后丢失的问题。
- 修复个人中心模型测试未读取草稿配置的问题。
- 修复空文本材料仍参与匹配的问题。
- 修复导师补全和模板改写失败原因未清晰展示的问题。
- 修复爬虫补全详情页时未充分尊重 Agent 判断的问题。
- 修复重复模板锚点可能被写入的问题。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.0.exe`。

- 发现新版本后，可以选择增量下载或全量下载。
