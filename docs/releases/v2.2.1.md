# v2.2.1

## 更新内容

### 新增功能

- 定时批量任务现在会处理已过期的发送批次，避免过期任务长期停留在待派发状态。
- 智能爬取 Token 相关界面会展示学校名称，便于区分不同学校的爬取任务和消耗来源。

### 体验优化

- 任务中心统一草稿审核入口，并补齐发送中状态展示，任务处理路径和当前进度更清楚。
- 任务页统一操作语义文案，并优化取消抓取任务后的重试提示，减少操作理解成本。
- 首页导师关系状态语义已修正，导师状态展示与后续筛选、跟进判断保持一致。
- 通信记录现在会更准确地显示往来内容和回复时间，便于回看导师沟通进展。

### 问题修复

- 修复批量发送中的并发竞态问题，降低定时或批量发送时出现异常状态的概率。
- 修复富文本导入后字体颜色、旧式相对字号和内联样式标记丢失的问题。
- 修复样式模板锚点改写保护过度的问题，让模板改写在保留格式的同时更稳定。
- 修复默认材料文本提取失败的问题，避免材料内容无法进入后续匹配和生成流程。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.1.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
