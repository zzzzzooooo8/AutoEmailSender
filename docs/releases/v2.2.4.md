# v2.2.4

## 更新内容

### 新增功能

- 批量发送支持随机均匀排程，任务会更自然地分散发送，减少集中投递带来的风险。
- 邮件回复同步改为按身份和教授持续增量同步，能更稳定地识别新的收件回复。
- 默认模板正文支持拖拽导入，准备批量任务时可以更快复用现有内容。
- 材料文件可通过系统应用以只读副本打开，查看附件和资料时更安全。

### 体验优化

- 应用更新提示会优先展示差量下载选项，升级时下载体积更小、等待时间更短。
- IMAP 配置提示更清晰，连接不同邮箱服务商时更容易完成设置。
- 后台任务会使用当前运行时配置启动，调整 worker 设置后无需依赖旧的环境默认值。
- 启动失败时会提供更详细的诊断信息，便于定位本地环境或配置问题。

### 问题修复

- 修复 IMAP 增量同步中部分回复可能未被拉取的问题。
- 修复同步回复时可能覆盖既有沟通记录的问题。
- 修复进程中断后任务状态未正确恢复的问题。
- 修复切换模型后身份历史记录可能丢失的问题。
- 修复身份邮箱重复保存时可能报错的问题。
- 修复仅被失败任务引用的材料无法删除的问题，并同步清理相关批量任务引用。
- 修复批量任务中已选择附件偶尔不稳定的问题。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.4.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
