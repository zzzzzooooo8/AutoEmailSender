# v2.2.5

## 更新内容

### 新增功能

- 桌面端支持设置开机自启动，常用设备可以更方便地保持应用可用。
- 卸载程序新增“清空本地数据”选项，需要彻底移除应用时可以一并清理本机数据。
- 测试邮件撰写页支持返回上一页，测试流程中的页面切换更顺手。

### 体验优化

- 桌面端后端启动更快，打开应用后的等待时间减少。
- 前端页面按需加载，首次进入和切换页面时更加轻快。
- 卸载时的数据清理时机更稳妥，降低卸载过程中误删或残留的风险。

### 问题修复

- 修复部分无附件邮件同步时正文解析异常的问题，邮件内容读取更稳定。
- 修复下载材料时偶尔打开空白窗口的问题，下载体验更符合预期。
- 修复删除材料后相关任务引用清理不完整的问题，减少无效材料残留。
- 修复恢复旧版已发送任务时状态不兼容的问题，历史任务展示更准确。
- 修复已发送任务的跟进状态可能丢失的问题，跟进记录会被正确保留。
- 修复部分前端和桌面端依赖配置问题，安装、构建和安全检查更稳定。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.5.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
