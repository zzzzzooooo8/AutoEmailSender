# v2.2.6

## 更新内容

### 新增功能

- 暂无面向普通用户的新功能，本次主要聚焦稳定性与体验改进。

### 体验优化

- 导师列表分页选择更顺手，跨页挑选导师时操作反馈更稳定。
- 撰写邮件草稿时会使用最新保存的模板内容，减少修改模板后仍套用旧内容的情况。

### 问题修复

- 修复 SMTP 连接测试失败时仍可能显示不准确结果的问题，邮箱配置检查更可信。
- 修复下载导师模板时偶尔打开空白窗口的问题，模板下载行为更符合预期。
- 修复已完成的批量任务可能阻止删除材料的问题，清理不再使用的材料更顺畅。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.6.exe`。

## 自动更新

- 应用内会自动检查更新。
- 发现新版本后，可以选择增量下载或全量下载。
