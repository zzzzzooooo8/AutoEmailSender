# v2.2.7

## 更新内容

### 新增功能

- 导师管理支持导出导师信息，导出的表格可用于备份、整理或再次导入。
- 邮件模板支持发送日期占位符，可在主题和正文中自动填入实际发送日期。
- 首页和导师管理页支持自定义每页显示数量，可在 `10`、`20`、`50` 条之间切换。
- 导师管理新增更完整的高级筛选能力，更容易按目标条件查找导师。

### 体验优化

- 发送日期会按本地时区显示，减少跨时区导致的日期偏差。
- AI 改写草稿时会保留用户已写好的日期，减少人工内容被误改的情况。

### 问题修复

- 修复审核定时批量草稿后可能丢失计划发送时间的问题，定时发送更可靠。
- 修复删除材料后测试撰写仍可能引用旧附件的问题，材料清理后草稿测试更准确。
- 修复导师管理筛选在边界条件下结果不准确的问题。
- 导出的导师表格会自动防护潜在公式风险，打开表格时更安全。

## 安装说明

- 普通用户只需下载 `AutoEmailSender Setup 2.2.7.exe`。

- 发现新版本后，可以选择增量下载或全量下载。
