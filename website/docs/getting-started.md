# 快速开始

Auto Email Sender 是一个本地运行的导师联系工具。安装版会自动启动本地后端服务，普通用户不需要安装 Python、Node.js、uv、npm 或 Git。

## 下载安装

1. 打开 [GitHub Releases](https://github.com/JunieXD/AutoEmailSender/releases)。
2. 进入最新版本，下载 `AutoEmailSender Setup x.y.z.exe`。
3. 双击安装，按安装向导完成安装。
4. 从开始菜单或桌面快捷方式打开 Auto Email Sender。

当前安装包暂未购买 Windows 代码签名证书。安装时如果看到「未知发布者」或 SmartScreen 提示，请确认下载来源是本项目 GitHub Releases 页面。

## 基本流程

第一次打开应用时，建议按系统内的「首次上手引导」完成配置：

1. [创建发件身份](./first-run#_1-创建发件身份)：填写发件人、邮箱、SMTP 和 IMAP。
2. [选择模型](./first-run#_2-配置模型)：配置 OpenAI 兼容模型，用于匹配分析和写信。
3. [准备材料](./first-run#_3-准备材料和邮件模板)：上传主材料，填写默认邮件主题和正文模板。
4. [补充导师](./first-run#_4-补充导师)：抓取、导入或手动维护导师信息。
5. [创建首个任务](./first-run#_5-创建首个任务)：选择导师，生成匹配分析和邮件任务。

完成首次配置后，日常使用通常是：

1. 在首页筛选导师，查看匹配情况。
2. 选择目标导师并创建联系任务。
3. 在工作区审核匹配结果和邮件草稿。
4. 确认无误后发送或安排定时发送。
5. 在任务中心查看批量任务、抓取任务和匹配分析任务。
6. 使用回复追踪关注后续回复。

## 更新

应用会检查 GitHub Releases 是否有新版本。有更新时会弹窗提示，确认后会下载新安装包并引导安装；如果当前已经是最新版本，会通过消息提示告知。
