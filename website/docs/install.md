# 安装 Windows 版

## 下载位置

Windows 安装版通过 GitHub Releases 发布：

[打开 GitHub Releases](https://github.com/JunieXD/AutoEmailSender/releases)

请下载最新版本中的 `AutoEmailSender Setup x.y.z.exe`。文件名里的 `x.y.z` 是版本号，例如 `0.1.0`。

## 安装步骤

1. 双击安装包。
2. 按安装向导选择安装位置。
3. 根据需要创建桌面快捷方式。
4. 安装完成后，从开始菜单或桌面快捷方式启动。

## 常见提示

如果 Windows 提示「未知发布者」或 SmartScreen 拦截，请先确认安装包来自本项目 GitHub Releases 页面。当前项目暂未购买 Windows 代码签名证书，因此首次安装时可能出现这类提示。

如果启动后看到白屏或连接失败，可以先退出应用并重新打开；如果仍然失败，请到 GitHub Issues 反馈问题，并附上系统版本、安装包版本和错误截图。
