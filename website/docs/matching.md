# 匹配分析

匹配分析用于辅助判断导师和你的材料是否匹配。它不是录取概率预测，也不能替代你自己的判断。

## 分析依据

匹配分析通常会参考：

- 你的主材料
- 导师研究方向
- 导师近期论文
- 学校、学院和职称等背景信息
- 当前选择的模型配置

如果导师缺少研究方向和近期论文，系统会提示缺少分析依据。

## 在哪里触发

可以在首页对单个导师或多个导师触发匹配分析。批量分析会创建匹配分析任务，进度可以在任务中心查看。

分析完成后，首页和工作区会展示匹配结果，帮助你决定是否创建联系任务。

## 如何使用结果

建议把匹配结果作为排序和筛选依据，而不是唯一判断依据。

你可以重点查看：

- 匹配分数
- 匹配理由
- 可能适合联系的研究方向
- 需要在邮件里强调的个人经历
- 不匹配或风险点

如果结果明显不符合实际，可以回到个人页检查主材料和模型配置，或回到导师管理页补充导师资料。
