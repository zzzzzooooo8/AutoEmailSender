# 导师管理

导师管理页用于准备和维护导师数据。导师数据是后续筛选、匹配分析、生成草稿和创建任务的基础。

## 智能抓取

智能抓取用于从学校官网整理导师信息。创建抓取任务时，通常需要提供学校、学院和起始页面地址。

抓取任务会在任务中心显示状态。抓取过程中可能产生候选导师，需要人工审核后才能进入正式导师库。

适合抓取的页面通常有这些特点：

- 页面本身是学院或课题组导师列表。
- 每位导师有独立详情页。
- 页面里包含邮箱、研究方向、论文或个人主页链接。

如果官网结构复杂，抓取结果可能需要人工审核和补充。

## 导入和维护

如果你已经有导师表格，可以先导入，再在导师管理页维护字段。

建议重点维护：

- 姓名
- 邮箱
- 学校
- 学院或院系
- 职称
- 研究方向
- 近期论文或代表成果
- 个人主页

导师资料越完整，匹配分析越可靠。缺少邮箱的导师不能直接发送邮件，缺少研究方向或论文的导师匹配分析会缺少依据。

## 审核候选导师

智能抓取产生的候选导师需要审核。审核时可以接受、拒绝或合并候选项。

建议先处理明显重复和明显无效的数据，再补充关键字段。这样后续在首页筛选和创建任务时会更清楚。
