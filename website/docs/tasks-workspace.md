# 任务与工作区

任务与工作区负责把「选择导师」变成可审核、可发送、可追踪的邮件流程。

## 创建任务

在首页选择导师后，可以创建联系任务。创建任务时需要确认：

- 目标导师
- 发件身份
- 模型配置
- 发送方式
- 是否立即发送或定时发送
- 每个发送窗口的时间和数量限制

如果你还没有完成首次配置，建议先回到个人页完成发件身份、模型、材料和模板设置。

## 发送设置

发送节奏在创建任务时设置，而不是在发件身份中设置。

目前有两种发送方式：

- **立即发送：** 任务创建后进入发送流程，适合少量导师或需要逐封确认的场景。
- **定时发送：** 选择发送日期、开始时间、结束时间和窗口内发送数量。系统会在指定时间窗口内动态发送，并按你设置的数量控制每天最多发送多少封。

如果选择定时发送，需要至少填写：

- 发送日期
- 发送开始时间
- 发送结束时间
- 窗口内发送数量

## 工作区审核

工作区用于查看匹配结果、邮件草稿和发送状态。

发送前建议逐封检查：

- 导师姓名和邮箱是否正确。
- 邮件主题是否自然。
- 正文是否贴合导师方向。
- 是否有明显模板痕迹。
- 附件和材料是否合适。
- 发送时间是否符合你的计划。

系统可以辅助生成草稿，但最终发送内容仍然需要你确认。

## 任务中心

任务中心用于查看后台任务和批量任务，包括：

- 批量发送任务
- 智能抓取任务
- 匹配分析任务

你可以在任务中心查看进度、暂停、恢复、重试或停止部分任务。具体可用操作取决于任务当前状态。

## 回复追踪

系统会通过 IMAP 检测导师回复，并在任务状态中体现。回复追踪依赖发件身份的 IMAP 配置，如果无法检测回复，请先回到个人页测试 IMAP。
